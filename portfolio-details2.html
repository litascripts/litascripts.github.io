Genset Voltage Monitoring System and Genset Measuring Pump with Camera

An embedded system using ESP32-CAM and sensors to remotely monitor generator voltage. Includes image capture and real-time voltage tracking.

Tools : Python, HTML+CSS, Flask, MySQL,
  ESP32-CAM, ESP8266, INA219 sensor,
  Arduino IDE, C/C++, MQTT, VSCode
Platform : Embedded + Web

A digital electronics-based genset monitoring system that uses INA219, ESP32 Cam, and ESP8266 sensors to collect, store, and upload data to the web Uses ESP32-cam, ESP8266, Flask, and MQTT to collect and visualize real-time readings.

The system uses ESP8266 to measure voltage and current using an INA219 sensor. To measures and monitors fuel using ESP32-Cam. Data is transmitted via MQTT and displayed on a live dashboard.
